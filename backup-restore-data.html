🔧 CorpManageSys Data Manager

Backup and restore your localStorage data

⚠️ Important Information:

This tool manages ALL your CorpManageSys data stored in browser localStorage. This includes companies, employees, tasks, users, templates, and activity logs. Always backup your data before restoring!

🌐 Same-Origin Requirement:

IMPORTANT: This tool MUST be accessed from your deployed domain (e.g., https://yourdomain.com/backup-restore-data.html) to work properly!

❌ Will NOT work: Opening this file locally (file:///...) or from a different domain

✅ Will work: Uploading this file to your public_html folder and accessing via your domain

To use this tool:

  1. Upload backup-restore-data.html to your server's public_html folder
  2. Access it via your domain: https://yourdomain.com/backup-restore-data.html
  3. You can now backup/restore localStorage data from your deployed application

💾 Backup Data

Download all CorpManageSys data from browser localStorage as a JSON file. This creates a complete backup that can be restored later.

📤 Restore Data

Upload a previously saved backup file to restore your data. Warning: This will replace ALL current data in your browser!

📊 View Current Data

See a summary of your current localStorage data without downloading.

🗑️ Clear All Data

Danger Zone: This will permanently delete ALL CorpManageSys data from this browser. This action cannot be undone!
